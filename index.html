Estrutura Básica

(Voltar)

A página principal do site é denominada index.html

<html> : tag que indica que o que está dentro da tag constitui um arquivo html. Tag é fechada com </html>

<head> : tag do cabeçalho. O que será carregado antes da página aparecer ao usuário.

<title> : Nome da página

<body> : tag do corpo. O que de fato aparece para o usuário


Tags

(Voltar)

Comandos utilizados para indicar ao navegador a estrutura do site

<i> : coloca o texto em itálicoOlha o exemplo

<strong> : coloca em negrito

<u>: sublinhado

<mark> : grifado

<sup> : elevado

<input> : tag que não tem um fechamento. <input type = “text”> : recebe um texto

<img> : recebe uma imagem. Também não tem fechamento

<br> : pula uma linha


Atributo das Tags

(Voltar)

Propriedades que as tags podem ter

id = “nome” : identificador

style = “nome” : serve para colocar comandos CSS diretamente no HTML

class = “nome” : forma de padronização com características pré definidas

type = “nome”: específico para o input, determina o tipo de valor recebido

src = “nome” : específico para img, representa o endereço da imagem

width = “num” : específico para img, determina a largura da imagem

heigth = “num” : específico para img, determina a altura da imagem

Listas Ordenadas e Não Ordenadas

(Voltar)

<ol> : tag da lista ordenada. <li> : item da lista

<ul> : tag da lista ordenada. <li> : item da lista

Exemplo de lista ordenada:

  1. Vc
  2. consegue
  3. sempre
  4. !!!
